# LangChain + Ollama Learning

This repository contains my experiments and learning projects using LangChain with Ollama for running Large Language Models locally.

The goal of this repo is to understand how to build AI applications using local LLMs, tools, agents, and retrieval systems.

## Tech Stack
- Python
- LangChain
- Ollama
- Jupyter Notebook
- Streamlit (for simple apps)

## What I'm Learning
- Running local LLMs with Ollama
- Using LangChain with local models
- Creating tools and agents
- Building simple RAG applications
- Connecting LLMs with APIs and databases
